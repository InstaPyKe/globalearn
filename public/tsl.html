Technical Specification Document (TSD)

Global Freelance & Rewards Platform Architecture

1. Professional Page Plan

A. The Public Facing Site (Landing & Trust)

  • Home Page: Hero section with CTA, "How it Works" guide, and real-time platform statistics.
  • Leaderboard: Live rankings of top earners and referrers to drive competition.
  • Proof of Payment Wall: A public feed of recent payouts (censored) to build global credibility.
  • Services/Features: Detailed breakdown of Ad-watching, Video tasks, and the Spin-to-Earn system.
  • FAQ & Help Center: Documentation on payment cycles, account rules, and troubleshooting.
  • Legal/Compliance: GDPR/CCPA compliant Terms of Service and Privacy Policy pages.

B. User Portal (The Earner’s Dashboard)

  • Main Dashboard: Real-time earnings snapshot, balance, and daily task progress.
  • The Earning Hub:
    • Ad-Watch & Video Lounge: High-performance players for YouTube/TikTok embeds.
    • Lucky Spin Room: Gamified UI with backend-validated prize logic.
  • Referral Management: Unique link generator and a 3-level visual downline tree.
  • Membership Tier Center: Upgrade paths (Basic/Gold/Platinum) for higher earning caps.
  • Wallet & Payouts: Full transaction history and multi-currency withdrawal requests.
  • Profile Settings: 2FA security, KYC document uploads, and language preferences.

C. Admin Portal (The Command Center)

  • Main Analytics: Global KPIs—total users, site-wide revenue vs. payout liability.
  • User Management: Identity verification queue (KYC) and account moderation tools.
  • Content & Ad Manager: Ad/Video uploader with pricing controls and daily view caps.
  • Financial Manager: Withdrawal approval queue and a Global Kill-Switch for emergencies.
  • Security Monitor: Real-time flags for IP/Device conflicts to prevent fraud.

2. The TLS Architecture Breakdown

Layer Recommended Technology Primary Responsibility
Presentation (Frontend) React.js / Next.js UI/UX, Multi-language support (i18n), and client-side task timers.
Logic (Backend API) Node.js / Python Referral commission logic, Anti-cheat filters, and API orchestration.
Data (Database) PostgreSQL / Redis User balances, transaction ledgers, and session caching.
The "Worker" Layer: Heavy tasks like calculating 3-level commissions (10% → 5% → 2%) are handled by background processors (Redis/BullMQ) to ensure the website remains fast.

3. Core Technical Modules & Security

  • Anti-Fraud Suite: Device fingerprinting and heartbeat checks to ensure tasks are completed by humans.
  • Global Payments: Support for Stripe, PayPal, and Crypto Gateways (USDT/BTC).
  • Scalability: Cloudflare CDN for global speed and DDoS protection.
  • Integrity: Immutable audit logs for every balance change and point-in-time database backups.

4. Final Checklist for Launch

  • Verification of legal links (Terms/Privacy) for payment processor approval.
  • Backend-only handling of "Spin" odds to prevent script manipulation.
  • Security of the cloudflared tunnel for the local development environment.
  • Taxation thresholds set for automated W-8BEN/W-9 collection.
